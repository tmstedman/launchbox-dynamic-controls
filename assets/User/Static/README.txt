Static — pre-made overlay images
==================================

PNG or JPG images placed here are used as-is for matching games, bypassing the
rendering pipeline entirely. Useful when you have a ready-made overlay image for
a specific game.

File naming: {Platform}/{Game}.png  or  {Platform}/{Game}.jpg
  Platform and game names must match LaunchBox's values exactly.

Example:
  User/Static/Arcade/1942.png
